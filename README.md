# 💰 Expense Tracker (Python)

A simple **Command-Line Expense Tracker** built using **Python** that allows users to record daily expenses, view all saved expenses, and calculate the total amount spent. Expense data is stored in a text file using Python file handling, making it easy to save and retrieve records.

---

## 📌 Features

- ➕ Add new expenses
- 📂 Store expenses in a text file
- 📋 View all recorded expenses
- 💰 Calculate total expenses
- 🖥️ Easy-to-use command-line interface
- 💾 Persistent data storage using file handling

---

## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/Siddha-W/Expense-Tracker.git
```

### 2. Open the Project Folder

```bash
cd Expense-Tracker
```

### 3. Run the Program

```bash
python expense_tracker.py
```

---

## 💡 How It Works

1. Run the Python program.
2. Choose **Add Expense** to enter:
   - Expense Name
   - Category
   - Amount
3. Choose **View Expenses** to display all saved expenses.
4. Choose **Show Total** to calculate the total amount spent.
5. Choose **Exit** to close the application.
